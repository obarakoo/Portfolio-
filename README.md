# Alfred M. Porcales - Portfolio

## 🚀 Overview
This repository contains the source code for my personal portfolio website, showcasing my work as a **QA & Front-End Developer**. Built with modern web technologies, it features projects ranging from web application development to QA automation.

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS with HSL color system
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Particles**: [@tsparticles/react](https://particles.js.org/)

## 📂 Project Structure
- `src/components/`: Reusable React components (Hero, Projects, Skills, etc.)
- `src/data/`: Data files for dynamic content (Skills)
- `src/profile-user/`: Images and profile assets
- `src/Resume/`: Professional resume (PDF)
- `index.html`: Main HTML entry point
- `styles.css`: Global styles and design system

## 🏃 Running Locally
1. **Clone the repository**
   ```bash
   git clone https://github.com/obarakoo/Portfolio-.git
   cd portfolio
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Start the development server**
   ```bash
   npm run dev
   ```

## 🌐 Deployment
The project is configured for deployment to GitHub Pages using the `gh-pages` package.
```bash
npm run deploy
```
